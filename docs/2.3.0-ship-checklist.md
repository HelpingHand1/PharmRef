- [ ] `npm run release:check`
- [ ] Confirm `Content validation: 15 diseases, 0 error(s), 0 warning(s)`

- [ ] Home loads with disease cards, quick actions, bookmarks, and recent views
- [ ] Search resolves shorthand terms such as `CAP`, `CAUTI`, `DFI`, `SAB`, and `CRE`

- [ ] Disease pages show review metadata and evidence cards
- [ ] Priority pathways do not show inherited-only metadata messaging unless intentionally expected
- [ ] Monograph pages open from home, search, disease overview, and empiric therapy links
- [ ] Evidence cards can copy citations and open either direct sources or PubMed fallback
- [ ] Compare flow selects two drugs, renders the side-by-side view, and resets cleanly
- [ ] Bookmarks can be added and removed from monograph and home surfaces
- [ ] Recent views reopen disease, subcategory, and monograph routes correctly
- [ ] Patient context updates regimen warnings for renal function, dialysis, pregnancy, and weight-sensitive agents
- [ ] Calculator deep links from monograph and regimen warnings open the calculators workspace
- [ ] Print preview hides `no-print` controls and keeps clinical content readable
- [ ] Hash routes load directly on refresh for disease, subcategory, monograph, compare, audit, and calculators

- [ ] Install prompt or standalone install flow still works
